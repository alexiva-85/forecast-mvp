# E5 — On-chain: CTF + Exchange (testnet)

**Статус:** draft закрыт для инфраструктуры и reference Amoy; Forecast-branded deploy — по желанию команды.

## Цель

Подготовить testnet-контур settlement (CTF + Exchange V2) параллельно off-chain MVP, без привязки к Polymarket CLOB для исполнения в РФ.

## Сделано в репозитории

1. **Vendor pin** — `contracts/VENDOR.lock` + `npm run onchain:vendor`
2. **Адреса Amoy (reference)** — Polymarket vanity deploy, сверены через RPC (`contracts/addresses/amoy-reference.json`)
3. **Деплой toolkit** — `npm run onchain:deploy:exchange` (обёртка upstream Foundry script)
4. **Smoke** — `npm run onchain:smoke` + vitest `tests/onchain-amoy.test.ts`
5. **Bridge-stub** — `src/lib/onchain/bridge-stub.ts` (типы ордера V2, без CLOB)
6. **Чеклист V2** — [v2-checklist.md](./v2-checklist.md)

## Архитектура (сейчас)

```text
Пользователь → Next.js → Supabase RPC (place_order)  ← источник истины MVP
                              │
                              └── (будущее) onchain bridge → Exchange.matchOrders
```
